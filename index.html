Pan and Zoom

There are two types of zooming, geometric and semantic. Geometric zooming is like zooming into an image. Individual shapes become bigger. Semantic zooming is where the shapes representing data points remain the same size.

You can click and drag to move the visualization. Scroll up/down to zoom in/out. Check out panAndZoom.js to see how it works.

Brushing

Allows you to select a range and move the window through a range. While you can zoom in and out in this example , the brushing refers to just dragging the window.

General Implementation for Interactions

  1. Create a static visualization.
  2. Set up inputs in HTML. You may have to populate options from the JS depending on the number of options and the type of interaction.
  3. Set up event listeners in the JS.
  4. Extract what the user selected.
  5. Filter data based on what the user selected. Any other preprocessing with the data.
  6. Update components of the visualization. Preferably using D3 data-joins.
